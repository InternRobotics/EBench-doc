  <p class="doc-intro">
    EBench（Elemental Mobile Manipulation Benchmark）是一个基于 NVIDIA Isaac Sim 的室内 VLA 操作基准，涵盖长程任务、灵巧操作以及移动操作，旨在公平高效地比较模型在更具挑战性的任务上的能力。
  </p>

  <p class="doc-intro">
    EBench 采用 client–server 架构。Server 端运行 Isaac Sim 并以黑盒的方式启动仿真环境；Client 端是一个依赖极少的轻量包，可以直接安装到模型所在的环境中，通过 import 通信 client 与 Server 交互，避免依赖冲突。
  </p>
