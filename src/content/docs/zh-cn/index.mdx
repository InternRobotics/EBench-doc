## Benchmark 范围

- 任务族：
- Pick-and-Place
- Long-Horizon
- Dexterous & Precise
- 当前设置中的泛化维度：
- 未见物体（unseen objects）
- 未见背景（unseen backgrounds）
- 未见指令（unseen instructions）
- 组合未见设置（`object + background + instruction`）

## Infra 组成

1. `EBench` 数据集：轨迹、评测任务和场景资产。
2. `GenManip-Sim` server：基于 Ray 的仿真评测调度。
3. `genmanip-client` 包：模型侧接入使用的 CLI 和 EvalClient API。

## 当前可提交的任务设置

- `ebench/mobile_manip/<split>`
- `ebench/table_top_manip/<split>`
- `ebench/generalist/<split>`
- `split` 取值：`{val_train, val_unseen, test}`

示例：

```bash
gmp submit ebench/mobile_manip/test
gmp submit ebench/table_top_manip/val_unseen
gmp submit ebench/generalist/val_train
```

## 端到端流程

1. 准备环境并安装 server/client 依赖。
2. 拉取 EBench 数据并映射到 `saved/`。
3. 启动 `ray_eval_server.py`。
4. 通过 `gmp submit` 提交一个或多个评测配置。
5. 运行 client worker（`gmp eval` 或自定义 EvalClient 循环）。
6. 用 `gmp status` 跟踪进度，并在 `saved/eval_results/` 查看结果。
