## 评测流程

1. 准备 Isaac Sim 兼容环境。
2. 下载或解压评测资产到 `saved/`。
3. 用 `ray_eval_server.py` 启动本地评测服务。
4. 用 `gmp submit` 提交评测任务。
5. 在 `saved/eval_results/` 中查看产物和结果。

## 仓库关键位置

- `configs/tasks/ebench/`：EBench 任务配置。
- `ray_eval_server.py`：本地 Ray 评测服务入口。
- `standalone_tools/packages/genmanip_client/`：`gmp` CLI 和客户端工具。
- `saved/`：资产、任务包和评测结果所在目录。
