## Evaluation flow

1. Set up an Isaac Sim compatible environment.
2. Download or unpack the benchmark assets into `saved/`.
3. Start the local evaluation server with `ray_eval_server.py`.
4. Submit an evaluation job with `gmp submit`.
5. Run local workers and inspect outputs under `saved/eval_results/`.

## Repository layout

- `configs/tasks/ebench/`: benchmark task configuration files.
- `ray_eval_server.py`: local Ray-backed evaluation server.
- `standalone_tools/packages/genmanip_client/`: the `gmp` CLI and reusable client utilities.
- `saved/`: downloaded assets, unpacked task packages, and evaluation results.
