  <p class="doc-intro">
    EBench (Embodied Benchmark) ist ein Indoor-VLA-Manipulations-Benchmark auf Basis von NVIDIA Isaac Sim.
    Er umfasst langfristige Aufgaben, geschickte Manipulation und mobile Manipulation und ermoeglicht
    einen fairen und effizienten Vergleich von Modellen bei anspruchsvollen Embodied-Aufgaben.
  </p>

  <p class="doc-intro">
    EBench verwendet eine Client-Server-Architektur. Der Server fuehrt Isaac Sim aus und stellt die
    Simulationsumgebung als Blackbox bereit. Ein leichtgewichtiges Client-Paket kann direkt in die
    Modellumgebung installiert und per Import eingebunden werden, um mit dem Server zu kommunizieren
    -- Abhaengigkeitskonflikte werden so vollstaendig vermieden.
  </p>
