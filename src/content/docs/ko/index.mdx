  <div class="doc-divider"></div>

  <p class="doc-intro">
    EBench(Embodied Benchmark)는 NVIDIA Isaac Sim 기반의 실내 VLA 조작 벤치마크로, 장기 과제, 정밀 조작, 이동 조작을 포괄하여 도전적인 임바디드 태스크에서 모델 성능을 공정하고 효율적으로 비교할 수 있도록 설계되었습니다.
  </p>

  <p class="doc-intro">
    EBench는 클라이언트-서버 아키텍처를 채택합니다. 서버는 Isaac Sim을 실행하며 시뮬레이션 환경을 블랙박스 형태로 제공합니다. 경량 클라이언트 패키지를 모델 환경에 직접 설치하고 import하여 서버와 통신할 수 있으므로, 의존성 충돌을 완전히 방지합니다.
  </p>
