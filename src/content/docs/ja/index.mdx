  <p class="doc-intro">
    EBench（Embodied Benchmark）は NVIDIA Isaac Sim 上に構築された室内 VLA マニピュレーションベンチマークです。ロングホライズンタスク、精密操作、モバイルマニピュレーションをカバーし、高難度の身体性タスクにおけるモデル性能を公平かつ効率的に比較できます。
  </p>

  <p class="doc-intro">
    EBench はクライアント・サーバーアーキテクチャを採用しています。サーバー側では Isaac Sim を実行し、シミュレーション環境をブラックボックスとして公開します。クライアント側は依存関係が極めて少ない軽量パッケージで、モデル側の環境に直接インストールして import するだけでサーバーと通信でき、依存関係の競合を完全に回避します。
  </p>
